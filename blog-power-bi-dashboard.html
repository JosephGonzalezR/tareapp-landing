Como hacer un Dashboard en Power BI: Guia paso a paso

Power BI Febrero 2026 12 min de lectura

Que es Power BI y por que aprenderlo

Power BI es la herramienta de inteligencia de negocios (Business Intelligence) de Microsoft que te permite transformar datos crudos en visualizaciones interactivas y reportes profesionales. Si estas estudiando ingenieria comercial, administracion, ingenieria civil industrial o cualquier carrera que involucre analisis de datos, aprender Power BI es una de las mejores inversiones que puedes hacer en tu formacion.

A diferencia de Excel, que trabaja bien con tablas estaticas, Power BI esta disenado especificamente para crear dashboards dinamicos que se actualizan en tiempo real. Esto significa que puedes conectar tus fuentes de datos una sola vez y el dashboard se actualizara automaticamente cada vez que los datos cambien.

Entre las principales ventajas de Power BI se encuentran:

  • Es gratuito: Power BI Desktop se descarga sin costo desde el sitio oficial de Microsoft.
  • Interfaz intuitiva: si ya manejas Excel, la curva de aprendizaje es mucho mas suave.
  • Conectividad: se conecta a cientos de fuentes de datos (Excel, CSV, bases de datos SQL, APIs, Google Sheets, etc.).
  • Demanda laboral: segun LinkedIn, Power BI es una de las habilidades mas demandadas en el area de datos en Latinoamerica.
  • DAX: su lenguaje de formulas te permite crear calculos complejos de forma eficiente.
Pro tip Si eres estudiante, puedes acceder a Power BI Pro gratis a traves del programa Microsoft Imagine Academy o con tu correo institucional .edu. Consulta con tu universidad si tienen convenio.

Requisitos previos e instalacion

Antes de empezar, necesitas tener lo siguiente preparado:

  • Un computador con Windows 10 o superior (Power BI Desktop no esta disponible nativamente para Mac, pero puedes usar una maquina virtual o la version web).
  • Al menos 4 GB de RAM (8 GB recomendados para datasets grandes).
  • Una cuenta de Microsoft (puede ser tu correo institucional).
  • Un archivo de datos de practica (Excel o CSV) con al menos 100 filas.

Descargar Power BI Desktop

Ve a powerbi.microsoft.com y haz clic en "Descargar gratis". Tambien puedes buscarlo directamente en la Microsoft Store de Windows.

Instalar el programa

Ejecuta el instalador y sigue las instrucciones. El proceso tarda entre 3 y 5 minutos dependiendo de tu conexion. No necesitas configuracion especial.

Iniciar sesion

Al abrir Power BI por primera vez, te pedira iniciar sesion con tu cuenta Microsoft. Usa tu correo institucional si tienes acceso a Power BI Pro.

Familiarizarte con la interfaz

Explora las tres vistas principales: Vista de informe (donde creas visualizaciones), Vista de datos (para inspeccionar tablas) y Vista de modelo (para relaciones entre tablas).

Conectar tus datos (Excel, CSV, bases de datos)

El primer paso real en Power BI es importar tus datos. Power BI soporta una enorme cantidad de fuentes, pero las mas comunes para trabajos universitarios son archivos de Excel y CSV.

Para conectar un archivo de Excel, sigue estos pasos:

  1. En la pantalla de inicio, haz clic en "Obtener datos" en la cinta de opciones.
  2. Selecciona "Libro de Excel" de la lista de conectores.
  3. Navega hasta tu archivo y seleccionalo.
  4. En el Navigator, marca las tablas u hojas que quieres importar.
  5. Haz clic en "Transformar datos" si necesitas limpiar los datos, o en "Cargar" si ya estan listos.
Buena practica Siempre usa "Transformar datos" antes de cargar. Ahi puedes revisar los tipos de datos (numeros, fechas, texto), eliminar filas vacias y renombrar columnas. Esto te ahorrara problemas mas adelante.

Si tus datos vienen de una base SQL, el proceso es similar: selecciona el conector de SQL Server, ingresa la direccion del servidor y el nombre de la base de datos, y Power BI se conectara directamente. Esto es especialmente util cuando trabajas con datasets grandes que no caben comodamente en un archivo de Excel.

Crear visualizaciones basicas

Una vez que tienes tus datos cargados, es hora de crear las visualizaciones que formaran tu dashboard. Power BI ofrece mas de 30 tipos de visualizaciones nativas, pero las mas usadas son:

Grafico de barras y columnas

Ideal para comparar valores entre categorias. Por ejemplo, si tienes datos de ventas por region, un grafico de barras es perfecto. Solo arrastra el campo de "Region" al eje y "Ventas" a los valores.

Grafico de torta o dona

Util para mostrar proporciones de un todo. Usalo cuando tienes entre 3 y 7 categorias. Con mas categorias, el grafico se vuelve dificil de leer y es mejor usar barras.

Grafico de lineas

Perfecto para mostrar tendencias a lo largo del tiempo. Si tienes datos mensuales o anuales, este es tu grafico. Arrastra la fecha al eje X y la metrica al eje Y.

Tarjetas (Cards)

Muestran un unico valor destacado, como el total de ventas, el promedio de calificaciones o el numero de registros. Son excelentes para KPIs en la parte superior del dashboard.

Para crear cualquier visualizacion, el proceso basico es:

  1. Haz clic en un area vacia del lienzo.
  2. En el panel de Visualizaciones (derecha), selecciona el tipo de grafico.
  3. Desde el panel de Campos, arrastra las columnas a las areas correspondientes (Eje, Valores, Leyenda).
  4. Ajusta los colores y formato desde la pestana de Formato del visual.

Un ejemplo practico: si tienes una tabla de ventas con columnas Producto, Region, Monto y Fecha, puedes crear una medida DAX para calcular el total de ventas:

Total Ventas = SUM('Ventas'[Monto])

Y para calcular el crecimiento porcentual mes a mes:

Crecimiento MoM =
VAR VentasMesActual = [Total Ventas]
VAR VentasMesAnterior =
    CALCULATE(
        [Total Ventas],
        DATEADD('Calendario'[Fecha], -1, MONTH)
    )
RETURN
    DIVIDE(
        VentasMesActual - VentasMesAnterior,
        VentasMesAnterior,
        0
    )
Tip DAX La funcion DIVIDE es mejor que usar el operador / directamente porque maneja automaticamente la division por cero, retornando el valor alternativo que le indiques (en este caso, 0).

Disenar el layout del dashboard

Un buen dashboard no es solo tener graficos bonitos, sino que la disposicion tenga logica y sea facil de leer. Aqui van algunos principios de diseno que puedes aplicar:

  • Jerarquia visual: los KPIs mas importantes van arriba (tarjetas con numeros grandes). Los graficos de tendencia en el medio. Los detalles y tablas abajo.
  • Regla del "Z": los usuarios leen en forma de Z. Coloca la informacion mas critica en la esquina superior izquierda.
  • Colores consistentes: usa una paleta de 3 a 5 colores maximo. Evita usar colores aleatorios para cada grafico.
  • Espacio en blanco: no llenes cada pixel del lienzo. El espacio en blanco ayuda a la lectura.
  • Titulo claro: agrega un titulo descriptivo y la fecha de actualizacion en la parte superior.

Para cambiar el fondo del lienzo y darle un look mas profesional, ve a Formato > Lienzo > Fondo y selecciona un color claro como #F5F5F5. Esto ayuda a que las tarjetas y graficos con fondo blanco resalten.

Tambien puedes agregar cuadros de texto, imagenes (como el logo de tu universidad) y formas rectangulares para crear secciones visuales dentro del dashboard. Usa Insertar > Cuadro de texto o Insertar > Imagen.

Agregar filtros e interactividad

Una de las grandes fortalezas de Power BI es la interactividad. Cuando haces clic en un segmento de un grafico, todos los demas graficos se filtran automaticamente. Esto se llama cross-filtering y funciona por defecto.

Ademas de esto, puedes agregar controles de filtro explicitos:

Slicers (segmentadores)

Son los filtros visuales que el usuario puede manipular. Para agregar uno, selecciona la visualizacion "Segmentacion de datos" y arrastra el campo que quieres usar como filtro (por ejemplo, "Region" o "Ano").

Los slicers pueden ser de tipo lista, desplegable, rango de fechas o deslizador numerico. Para un dashboard limpio, te recomiendo usar el estilo desplegable que ocupa menos espacio.

Filtros de pagina y de informe

En el panel de Filtros puedes agregar filtros que aplican a toda la pagina o a todo el informe. Estos son utiles cuando quieres que ciertos datos no aparezcan nunca (por ejemplo, excluir registros de prueba).

Bookmarks (marcadores)

Los bookmarks te permiten guardar el estado actual de los filtros y la visibilidad de los elementos. Puedes crear botones que al hacer clic cambien la vista del dashboard. Por ejemplo, un boton que muestre "Vista Resumen" y otro que muestre "Vista Detallada".

Para configurar el cross-filtering entre graficos, selecciona un visual, ve a Formato > Editar interacciones y elige si los demas graficos deben filtrarse, resaltarse o no verse afectados.

-- Ejemplo de medida DAX con filtro contextual
Ventas Filtradas =
CALCULATE(
    SUM('Ventas'[Monto]),
    FILTER(
        'Productos',
        'Productos'[Categoria] = "Electronica"
    )
)

-- Medida con ALL para ignorar filtros
Porcentaje del Total =
DIVIDE(
    [Total Ventas],
    CALCULATE([Total Ventas], ALL('Ventas')),
    0
)

Publicar y compartir tu dashboard

Una vez que tu dashboard esta listo, tienes varias opciones para compartirlo:

Publicar en el servicio de Power BI

Haz clic en "Publicar" en la cinta de opciones. Selecciona un espacio de trabajo y tu informe estara disponible en la nube. Desde ahi puedes generar un enlace para compartir.

Exportar a PDF

Si tu profesor necesita un archivo estatico, ve a Archivo > Exportar > PDF. Cada pagina del informe se convertira en una pagina del PDF.

Exportar a PowerPoint

Tambien puedes exportar a PPTX para incluir las visualizaciones en una presentacion. Cada visual se exporta como imagen con un enlace al informe interactivo.

Compartir el archivo .pbix

Si tu profesor usa Power BI, simplemente envia el archivo .pbix por correo o plataforma del curso. Es la opcion mas sencilla para trabajos universitarios.

Ojo con el peso del archivo Los archivos .pbix pueden pesar bastante si tienen muchos datos importados. Si tu archivo pesa mas de 25 MB, considera usar DirectQuery en vez de Import para la conexion de datos, o reduce las filas que importas.

Conclusion y proximos pasos

Ya tienes las bases para crear un dashboard profesional en Power BI. Recapitulando lo que aprendiste:

  1. Que es Power BI y por que es una herramienta clave para el analisis de datos.
  2. Como instalar y configurar el entorno de trabajo.
  3. Como conectar distintas fuentes de datos.
  4. Como crear visualizaciones basicas y escribir formulas DAX.
  5. Principios de diseno para un dashboard efectivo.
  6. Como agregar interactividad con filtros, slicers y bookmarks.
  7. Como publicar y compartir tu trabajo final.

Para seguir avanzando, te recomiendo explorar estos temas:

  • Power Query avanzado: transformaciones complejas de datos antes de visualizarlos.
  • DAX intermedio: funciones de inteligencia de tiempo como TOTALYTD, SAMEPERIODLASTYEAR.
  • Row-Level Security (RLS): para controlar que datos puede ver cada usuario.
  • Power BI con R o Python: para crear visualizaciones personalizadas con codigo.
